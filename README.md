# Agent Skills

Claude Code skills for Jira workflow automation and sprint planning.

## Skills

### jira
Manage Jira issues with full workflow support — search, create, update, transition, link, and batch operations. Includes team structure, custom field mappings, and classification.

### sprint-planner
Prepare and organize a new sprint for the Team Lead. Collects inputs, ensures the sprint exists, creates a support bucket story, calculates capacity, and outputs a summary with links.
